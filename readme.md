# Device Library

Console application for managing a catalog of electronic devices and their specifications.

---

## Overview

Application built to store, view, edit, delete, and search device data using a structured model.

---

## Features

* Add device
* View devices
* Edit device
* Delete device
* Search by model or manufacturer
* Multiple hardware parameters per device

---

## Tech Stack

* C# (.NET)
* Console UI
* JSON data storage

---

## Architecture

* MVVM-like separation
* Interface-based logic
* Service layer for search and input handling

---

## Data Model

Structured device data includes:

* Device info
* Display info
* Hardware parameters
* Software info

---

## Limitations

* No sorting
* Basic search
